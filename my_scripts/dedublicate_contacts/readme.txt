Спасибо slavos1sss - автору статьи https://habr.com/ru/articles/475780/ за базовую информацию по обработке формата vCard.

Файлы с визитками разных версий (2.1 и 3) должны быть в разных итоговых файлах, иначе они не принимаются смартфоном. Разделить как нибудь вручную или дописать скрипт.

Для объединения нескольких .vcf файлов в один можно запустить скрипт all_in_one.py.
